== Introduction

atcoder tool for Gradle

== Getting Started

.build.gradle.kts
[source, kotlin]
----
plugins {
    id("com.nisecoder.gradle.atcoder")
}
----

.setting.gradle.kts
----
include::src/getting-started/how-to-apply-plugin.gradle.kts[]
----

credentials use values saved with
https://github.com/etiennestuder/gradle-credentials-plugin[gradle-credential-plugin]

----
$ gradle  addCredentials --key atcoder.username --value {user atcoder username}
$ gradle  addCredentials --key atcoder.password --value {user atcoder password}
----
